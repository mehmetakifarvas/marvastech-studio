## ✨ Features
- ✅ AVR support (ATmega)
- ✅ ARM support (STM32)
- ✅ Real-time serial port plotter
- ✅ One-click compile and upload
- ✅ Code editor with syntax highlighting
- ✅ Portable - runs from USB
